Starknet API method that returns the storage value at a given position for a given account.

It provides the value stored at a specific position in the storage of an account. This is useful for clients or wallets to inspect the state of a smart contract at a specific block.

## Parameters

* `address` — the string of the address of the account.
* `position` — the string of the position in the storage.

## Response

* `result` — the value stored at the given position in the account's storage, represented as a hexadecimal string.

## Use case

A practical use case for `starknet_getStorageAt` is to inspect the state of a smart contract. For example, if a smart contract stores the balance of an account at a specific position in its storage, you can use `starknet_getStorageAt` to retrieve the balance of the account at a specific block.
