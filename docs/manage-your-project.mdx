 title: "Manage your projects"
---

 ## Create a project

 You must create a <Tooltip tip="A project organizes your blockchain networks and nodes.">project</Tooltip> to be able to deploy a node or a network, or to join a network.

 To create a project:
 1. Log in to your [Chainstack account](https://console.chainstack.com/).
2. Click **Create project**.
3. Provide **Project name**
4. Provide **Description** (optional).
5. Click **Create**.

 ## Add a network to a project

 A <Tooltip tip="A project organizes your blockchain networks and nodes.">project</Tooltip> can have more than one network.

 To add a network to a project, see [Join a public network](/docs/manage-your-networks#join-a-public-network).

 ## Edit a project

 To edit a project:
 1. Click your project.
2. Click **Edit**.
3. Edit your **Project name** and/or **Description**.
4. Click **Save** to apply changes.

 ## Delete a project

 <Warning>
   To be able to delete a project, you must first delete all nodes associated with the project. See [Delete a node](/docs/manage-your-node#delete-a-node).
 </Warning>

 To delete a project:
 1. Click your project.
2. Click **Edit** > **Delete**.
3. Click **Yes, I'm sure** to confirm the action.
