## JSON-RPC API

Interact with your Bitcoin nodes using [JSON-RPC API](https://en.bitcoin.it/wiki/API_reference_\(JSON-RPC\)#JSON-RPC).

Use [curl](https://curl.haxx.se) or [Postman](https://www.getpostman.com) to invoke [Bitcoin API methods](https://bitcoin.org/en/developer-reference#bitcoin-core-apis).

Example below demonstrates how to get basic network information from your Bitcoin node HTTPS endpoint:

<CodeGroup>
  ```bash cURL
  curl YOUR_CHAINSTACK_ENDPOINT -d '{"method":"getblockchaininfo","params":[],"id":1}'
  ```
</CodeGroup>

where YOUR\_CHAINSTACK\_ENDPOINT is your node HTTPS endpoint protected either with the key or password

## Python

Work with Bitcoin from your Python application.
<Steps>
  <Step>
    Install [bitcoincli](https://github.com/chainstack/bitcoincli).
  </Step>
  <Step>
   Configure the client to use `host` and `port` of the node HTTPS endpoint, the corresponding node `username`, and `password`:

   <CodeGroup>
     ```python Python
     from bitcoincli import Bitcoin

     host = "nd-123-456-789.p2pify.com"
     port = "443"
     username = "user-name"
     password = "pass-word-pass-word-pass-word"

     bitcoin = Bitcoin(username, password, host, port)
     ```
   </CodeGroup>
  </Step>
  <Step>
    Invoke any methods from the [Bitcoin API specification](https://bitcoin.org/en/developer-reference#bitcoin-core-apis):

   <CodeGroup>
     ```python Python
     info = bitcoin.getblockchaininfo()
     print(info)
     ```
   </CodeGroup>
  </Step>
</Steps>
