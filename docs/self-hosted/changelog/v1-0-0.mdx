**Status:** Beta

This release includes the full deployment stack for Ethereum full nodes on your own Kubernetes infrastructure.

## Features

- **Ethereum Mainnet support** — deploy full nodes using Reth execution client and Prysm consensus client
- **Ethereum testnets support** — Sepolia and Hoodi testnets
- **Web-based control panel** — intuitive interface for node deployment and management
- **Kubernetes-native architecture** — Helm-based deployment on any Kubernetes distribution (k3s, k8s, EKS, GKE, AKS)

## Known limitations

- Single protocol support (Ethereum only)
- Full sync only (no snapshot-based deployment)
- No built-in monitoring dashboards
- No archive node support
- No validator support
