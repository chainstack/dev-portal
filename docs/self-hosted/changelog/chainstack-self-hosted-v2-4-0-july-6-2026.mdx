This release adds Polygon support, rounds out node management with renaming and has several small display fixes.

## Polygon support

Polygon is now available as a deployment preset, running Bor and Heimdall together as a matched pair. It shows up in the new-node flow like any other supported network, so bringing up a Polygon node follows the same one-click experience as other chains.

## Node management

- **Node renaming** — nodes can now be renamed after deployment instead of having to be recreated to fix a name.
- **Cleaner protocol and network names** — network labels in the console no longer repeat the protocol name.
- **Logo fixes** — logos now resolve correctly for protocols with multi-word names.

## Fixes

- Fixed an issue where `cpctl` could report an unknown command in certain cases.
