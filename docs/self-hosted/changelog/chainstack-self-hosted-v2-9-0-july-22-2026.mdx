This release adds Robinhood Chain Sepolia as a new supported test network.

## Robinhood Chain Sepolia support

Robinhood Chain Sepolia is now available as a deployment target, built on the Arbitrum Nitro preset. You can deploy and manage Robinhood Chain Sepolia nodes from the Control Panel with the same workflows as other supported networks.
