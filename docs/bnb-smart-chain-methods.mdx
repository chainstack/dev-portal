---

See also [interactive BNB Smart Chain API call examples](/reference/getting-started-bnb-chain).

<Warning>
  **BSC limitation**: On Binance Smart Chain, `eth_getProof` currently only works with the `"latest"`, `"earliest",` `"safe"`, `"finalized"`, `"pending"` block tags. Using specific block numbers or other tags (`"safe"`, `"finalized"`, `"pending"`) will result in an error. This limitation will be addressed in a future update.
</Warning>
