  <Accordion title="Overview">

  This script shows you how to use `eth_call` to simulate a swap from ETH to an ERC-20 token.

  </Accordion>
  <Accordion title="Prerequisites">

  Following this recipe, you will need a Chainstack account and an Ethereum node.

  [Sign up with Chainstack](https://console.chainstack.com/user/account/create).
  [Deploy a node](https://docs.chainstack.com/docs/manage-your-networks#join-a-public-network).
  [View node access and credentials](https://docs.chainstack.com/docs/manage-your-node#view-node-access-and-credentials).

  </Accordion>
  <Accordion title="Environment setup">

  Install [node.js](https://nodejs.org/en/) if it has not been installed yet.

  Create a new directory for your project, then install the `web3.js` library.

  `npm install web3`

  </Accordion>

  <Accordion title="The main function">

  The `simulateBuy` function simulates the process of buying tokens on Uniswap V2 using ETH without actually making a blockchain transaction. Given current market conditions, it calculates the number of tokens you would receive for a specified amount of ETH using `eth_call`.

  ### Parameters
  - `ethAmount`: The amount of ETH you intend to swap for tokens.
  - `tokenAddress`: The token's contract address you want to buy.

  ### Steps

  <Steps>
    <Step title="Setup">
      Establishes a deadline (20 minutes from the current time) and defines the swap path from Wrapped ETH (WETH) to the desired token.
    </Step>
    <Step title="Token Information">
      Token Information: This function retrieves the name and symbol of the target token using its contract, enhancing the readability of the simulation's output.
    </Step>
    <Step title="Transaction Encoding">
      Encodes the `swapExactETHForTokens` method call of the Uniswap Router contract with:
    - `amountOutMin` set to 0 (for simplicity, indicating no minimum token amount is required).
    - The swap path (WETH to the desired token).
    - The recipient's address (where to send the tokens).
    - The swap deadline.
    </Step>
    <Step title="Simulation">
      Simulates the transaction using `web3.eth.call` with the encoded data. This call executes the transaction logic without making state changes or incurring gas costs.
    </Step>
    <Step title="Result Processing">
      Decodes the simulation result to extract and convert the swapped token amounts into readable formats. Prints the simulation details, including the token name, symbol, and the amounts of ETH and tokens involved in the simulated swap.
    </Step>
    <Step title="Error Handling">
      Catches and logs any errors encountered during the simulation process, such as failed calls to the blockchain or issues with the contract ABIs or addresses.
    </Step>
  </Steps>

  ### UsageIdeal for testing and development purposes, allowing developers to:

  - Estimate the outcome of a swap operation without spending real ETH.
  - Debug and optimize smart contract interactions in a risk-free environment.

  ### Implementation Considerations
  - **ABI and Address Accuracy**: Ensure the ABIs and contract addresses used are correct and up-to-date to avoid simulation inaccuracies.
  - **Network Connection**: Requires a connection to an Ethereum node. The `NODE_URL` must be correctly set up for the simulation to interact with the Ethereum network.
  - **Deadline Handling**: The hardcoded deadline may need adjustment based on the context of use, especially in scenarios with network congestion.

  This code is a powerful tool for developers to understand and demonstrate token-swapping mechanics on Uniswap V2, providing insights into the process and outcomes without requiring live transactions.

  ```js index.js
  */
  async function simulateBuy(ethAmount, tokenAddress) {
    const deadline = Math.floor(Date.now() / 1000) + 60 * 20; // 20 minutes from the current time
    const path = [WETH_ADDRESS, tokenAddress]; // Trading path (WETH -> Token)

    // Get token name and symbol
    const tokenName = await tokenContract.methods.name().call();
    const tokenSymbol = await tokenContract.methods.symbol().call();

    // Encode the swapExactETHForTokens method call
    const data = routerContract.methods.swapExactETHForTokens(
      web3.utils.toWei('0', 'ether'), // amountOutMin
      path,
      DESTINATION, // Recipient address
      deadline
    ).encodeABI();

    const tx = {
      to: uniswapRouterAddress,
      data: data,
      value: web3.utils.toWei(ethAmount.toString(), 'ether')
    };

    try {
      const result = await web3.eth.call(tx);
      const decodedOutput = web3.eth.abi.decodeParameters(['uint256[]'], result);

      const [ethAmount, tokenAmount] = decodedOutput[0];

      const ethAmountConverted = web3.utils.fromWei(ethAmount, 'ether');
      const tokenAmountConverted = web3.utils.fromWei(tokenAmount, 'ether');

      console.log('Buy simulation success');
      console.log(`Token name: ${tokenName}`);
      console.log(`Token symbol: ${tokenSymbol}`);
      console.log(`ETH Amount: ${ethAmountConverted} ETH`);
      console.log(`Token Amount: ${Number(tokenAmountConverted).toFixed(3)} ${tokenSymbol}`);
    } catch (error) {
      console.error('Buy simulation failed:', error.message);
    }
  }
  ```

/**
 * Simulates buying tokens using Uniswap V2 Router
 * @param {string} ethAmount - The amount of ETH to spend for buying tokens
 * @param {string} tokenAddress - The address of the token to buy
 */
async function simulateBuy(ethAmount, tokenAddress) {
  const deadline = Math.floor(Date.now() / 1000) + 60 * 20; // 20 minutes from the current time
  const path = [WETH_ADDRESS, tokenAddress]; // Trading path (WETH -> Token)

  // Get token name and symbol
  const tokenName = await tokenContract.methods.name().call();
  const tokenSymbol = await tokenContract.methods.symbol().call();

  // Encode the swapExactETHForTokens method call
  const data = routerContract.methods.swapExactETHForTokens(
    web3.utils.toWei('0', 'ether'), // amountOutMin
    path,
    DESTINATION, // Recipient address
    deadline
  ).encodeABI();

  const tx = {
    to: uniswapRouterAddress,
    data: data,
    value: web3.utils.toWei(ethAmount.toString(), 'ether')
  };

  try {
    const result = await web3.eth.call(tx);
    const decodedOutput = web3.eth.abi.decodeParameters(['uint256[]'], result);

    const [ethAmount, tokenAmount] = decodedOutput[0];

    const ethAmountConverted = web3.utils.fromWei(ethAmount, 'ether');
    const tokenAmountConverted = web3.utils.fromWei(tokenAmount, 'ether');

    console.log('Buy simulation success');
    console.log(`Token name: ${tokenName}`);
    console.log(`Token symbol: ${tokenSymbol}`);
    console.log(`ETH Amount: ${ethAmountConverted} ETH`);
    console.log(`Token Amount: ${Number(tokenAmountConverted).toFixed(3)} ${tokenSymbol}`);
  } catch (error) {
    console.error('Buy simulation failed:', error.message);
  }
}
