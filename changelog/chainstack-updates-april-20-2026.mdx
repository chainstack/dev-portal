**MCP Server**. New tool: **`get_chainstack_pricing`**. Ask your agent anything about Chainstack pricing and get an answer grounded in live data — no guessing, no stale quotes.

* Covers plan tiers (Developer through Enterprise and Pay As You Go), the feature matrix, add-on pricing (Unlimited Node, Yellowstone gRPC, Warp transactions), and the full per-chain dedicated-node catalog (~87 SKUs with regions, hardware flavors, and hourly/monthly USD prices).
* No API key required. Every call fetches fresh from the public pricing sources — the server holds zero hardcoded pricing, so changes on chainstack.com show up immediately.
* Example prompts: *"I expect 50M RU/month and need archive data — which plan fits?"* or *"How much is a dedicated Ethereum archive node in London?"*
