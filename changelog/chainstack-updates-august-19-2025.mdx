title: "Chainstack updates: August 19, 2025"
description: "Starknet nodes upgraded to Pathfinder v0.19.0 — the default JSON-RPC API version served on the root endpoint changed from 0.7.1 to 0.8.0."
---

**Starknet**. Chainstack upgraded Starknet nodes to [Pathfinder v0.19.0](https://github.com/eqlabs/pathfinder/releases/tag/v0.19.0). The default JSON-RPC API version served on the `/` path changed from 0.7.1 to 0.8.0. To keep the previous version, pin `/rpc/v0_7` in your endpoint URL. See [Pathfinder default API version change 0.19.0](/reference/starknet-pathfinder-default-api-version-change) for details.
