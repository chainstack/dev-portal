Plasma API method that returns the RLP-encoded transaction.

## Parameters

* `data` — the hash of the transaction

## Response

* `data` — the RLP-encoded transaction

## `debug_getRawTransaction` code examples

<CodeGroup>
```javascript ethers.js
const { JsonRpcProvider } = require("ethers");

const provider = new JsonRpcProvider("CHAINSTACK_NODE_URL");

async function call() {
  const result = await provider.send("debug_getRawTransaction", [
    "0xTRANSACTION_HASH"
  ]);
  console.log(result);
}

call();
```
