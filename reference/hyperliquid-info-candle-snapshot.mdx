## Parameters

* `type` (string, required) ‚Äî Must be `"candleSnapshot"`
* `req` (object, required) ‚Äî Request parameters:
  * `coin` (string, required) ‚Äî Asset identifier ("BTC", "ETH" for perpetuals; "@107" for spot)
  * `interval` (string, required) ‚Äî Candle interval (see supported intervals below)
  * `startTime` (integer, required) ‚Äî Start time in milliseconds (epoch timestamp)
  * `endTime` (integer, required) ‚Äî End time in milliseconds (epoch timestamp)

## Returns

Returns an array of candlestick objects with OHLCV data:

* `t` ‚Äî Open time timestamp (milliseconds)
* `T` ‚Äî Close time timestamp (milliseconds)
* `o` ‚Äî Open price (string)
* `h` ‚Äî High price (string)
* `l` ‚Äî Low price (string)
* `c` ‚Äî Close price (string)
* `v` ‚Äî Volume traded (string)
* `n` ‚Äî Number of trades (integer)
* `i` ‚Äî Interval (string)
* `s` ‚Äî Symbol (string)

<Warning>
Only the most recent 5000 candles are available. Older data beyond this limit is not accessible.
</Warning>

## Supported intervals

`"1m"`, `"3m"`, `"5m"`, `"15m"`, `"30m"`, `"1h"`, `"2h"`, `"4h"`, `"8h"`, `"12h"`, `"1d"`, `"3d"`, `"1w"`, `"1M"`

```python Python (hyperliquid-python-sdk)
from hyperliquid.info import Info
from hyperliquid.utils import constants

info = Info(constants.MAINNET_API_URL, skip_ws=True)

candles = info.candles_snapshot(
    name="BTC",
    interval="1h",
    startTime=1754300000000,
    endTime=1754400000000,
)
print(candles)
```

```typescript TypeScript (@nktkas/hyperliquid)
import * as hl from "@nktkas/hyperliquid";

const transport = new hl.HttpTransport();
const client = new hl.InfoClient({ transport });

const candles = await client.candleSnapshot({
  coin: "BTC",
  interval: "1h",
  startTime: 1754300000000,
  endTime: 1754400000000,
});
console.log(candles);
```
