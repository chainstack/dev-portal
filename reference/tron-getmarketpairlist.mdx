TRON API method that retrieves all available trading pairs on the TRON DEX marketplace. This provides a comprehensive list of all token pairs that can be traded on the decentralized exchange.

## Parameters

This method does not require any parameters.

## Response

* `orderPair` — array of trading pair objects, each containing:
  * `sell_token_id` — token ID for the sell side of the pair (empty for TRX)
  * `buy_token_id` — token ID for the buy side of the pair (empty for TRX)

## Use case

The `wallet/getmarketpairlist` method is used for:

* Discovering all available trading pairs on TRON DEX.
* Building trading interface dropdown menus and pair selection lists.
* Analyzing market coverage and available token combinations.
* Implementing comprehensive market monitoring across all pairs.
* Creating trading pair validation for order placement systems.
* Building market overview dashboards showing all tradeable assets.

<Note>
Trading pairs on TRON DEX consist of combinations between TRX and TRC-10 tokens. Empty token IDs represent TRX. Each pair represents a market where users can exchange one asset for another through the decentralized order book system.
</Note>
