# Get Vesting Contracts

The `vesting` endpoint retrieves information about vesting contracts based on specified filters. This allows you to find vesting contracts associated with specific addresses or wallets.

## Parameters

* `contract_address` (array of strings) — Vesting contract address in any form.
* `wallet_address` (array of strings) — Wallet address to filter by owner or sender.
* `check_whitelist` (boolean) — Check if wallet address is in whitelist.
* `limit` (integer) — Limit number of queried rows.
* `offset` (integer) — Skip first N rows.

## Use case

The `vesting` endpoint is useful for:

1.  **Portfolio Trackers:** To identify and display vesting schedules for users.
2.  **Vesting Dashboards:** To show the status of token unlocks for investors or team members.
3.  **Auditing Tools:** To verify the existence and parameters of vesting contracts.
