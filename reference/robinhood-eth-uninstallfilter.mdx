Robinhood Chain API method that uninstalls a filter with the given ID.

## Parameters

* `quantity` — the filter ID

## Response

* `boolean` — `true` if the filter was successfully uninstalled, `false` otherwise

## `eth_uninstallFilter` code examples

<CodeGroup>
```javascript ethers.js
const { JsonRpcProvider } = require("ethers");

const provider = new JsonRpcProvider("CHAINSTACK_NODE_URL");

async function call() {
  const result = await provider.send("eth_uninstallFilter", ["0x1"]);
  console.log(result);
}

call();
```

```python web3.py
from web3 import Web3

node_url = "CHAINSTACK_NODE_URL"
web3 = Web3(Web3.HTTPProvider(node_url))

result = web3.provider.make_request("eth_uninstallFilter", ["0x1"])
print(result)
```

```shell cURL
curl -X POST "CHAINSTACK_NODE_URL" \
  -H "Content-Type: application/json" \
  --data '{
    "jsonrpc": "2.0",
    "method": "eth_uninstallFilter",
    "params": ["0x1"],
    "id": 1
  }'
```
