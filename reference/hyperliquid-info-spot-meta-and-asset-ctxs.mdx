The `info` endpoint with `type: "spotMetaAndAssetCtxs"` retrieves comprehensive metadata and market contexts for all spot assets on the Hyperliquid exchange. This endpoint provides detailed information about token specifications, trading pairs, market prices, volumes, supply metrics, and EVM contract details for spot trading analysis and integration.

## Parameters

### Request body

* `type` (string, required) ‚Äî The request type. Must be `"spotMetaAndAssetCtxs"` to retrieve spot market metadata and asset contexts.

## Response

The response returns an array with two elements:
1. **Token metadata array** ‚Äî Detailed specifications for all spot tokens
2. **Market context array** ‚Äî Real-time trading data and supply information

### Token Metadata Structure

Each token metadata object contains:

* `name` (string) ‚Äî Token symbol/ticker (e.g., "PURR", "HYPE")
* `szDecimals` (number) ‚Äî Size decimals for trade sizing precision
* `weiDecimals` (number) ‚Äî Wei decimals for on-chain representation
* `index` (number) ‚Äî Unique index identifier for the token
* `tokenId` (string) ‚Äî Hexadecimal token identifier
* `isCanonical` (boolean) ‚Äî Whether this is the canonical version of the token
* `evmContract` (object/null) ‚Äî EVM contract details if bridged:
  * `address` (string) ‚Äî Contract address on EVM chain
  * `evm_extra_wei_decimals` (number) ‚Äî Additional decimal adjustment for EVM
* `fullName` (string/null) ‚Äî Full descriptive name of the token
* `deployerTradingFeeShare` (string) ‚Äî Fee share percentage for token deployer

### Market Context Structure

Each market context object contains:

* `prevDayPx` (string) ‚Äî Previous day's closing price
* `dayNtlVlm` (string) ‚Äî Daily notional volume in USDC
* `markPx` (string) ‚Äî Current mark price
* `midPx` (string) ‚Äî Current mid price from order book
* `circulatingSupply` (string) ‚Äî Circulating token supply
* `coin` (string) ‚Äî Trading pair identifier (e.g., "PURR/USDC", "@150")
* `totalSupply` (string) ‚Äî Total token supply
* `dayBaseVlm` (string) ‚Äî Daily base asset volume

### Trading Pair Notation

* **Named pairs** ‚Äî Direct symbol notation (e.g., "PURR/USDC")
* **Index pairs** ‚Äî Reference by index (e.g., "@150" for token index 150)
* **Base pair** ‚Äî Most pairs trade against USDC as the quote currency

### Notes

* Token indices provide a stable reference for programmatic access
* Mark prices may differ from mid prices during volatile market conditions
* Circulating supply may be less than total supply due to locked or vested tokens
* EVM contract details enable cross-chain asset tracking and arbitrage
* Daily volumes reset at UTC midnight
* Some tokens may have null full names or EVM contracts if not applicable
