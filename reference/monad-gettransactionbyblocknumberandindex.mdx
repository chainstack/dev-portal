Monad API method that returns information about a transaction by block number and transaction index position. This method allows you to retrieve a specific transaction using the block number and the transaction's position within that block.

## Parameters

* `quantity|tag` ‚Äî the block number as a hexadecimal string, or one of the following block tags:
  * `latest` ‚Äî the most recent block in the canonical chain
  * `earliest` ‚Äî the genesis block
  * `pending` ‚Äî the pending state/transactions
* `quantity` ‚Äî the transaction index position within the block, encoded as hexadecimal.

## Response

* `result` ‚Äî the transaction object, or `null` when no transaction was found:
  * `blockHash` ‚Äî hash of the block containing this transaction
  * `blockNumber` ‚Äî number of the block containing this transaction
  * `from` ‚Äî address of the sender
  * `gas` ‚Äî gas provided by the sender
  * `gasPrice` ‚Äî gas price in wei
  * `hash` ‚Äî hash of the transaction
  * `input` ‚Äî the data sent along with the transaction
  * `nonce` ‚Äî number of transactions made by the sender
  * `to` ‚Äî address of the receiver
  * `transactionIndex` ‚Äî integer of the transaction's index position
  * `value` ‚Äî value transferred in wei
  * `v`, `r`, `s` ‚Äî signature values
