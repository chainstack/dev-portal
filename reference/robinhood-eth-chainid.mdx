Robinhood Chain API method that returns the chain ID of the current network.

## Parameters

None

## Response

* `quantity` — the chain ID

## `eth_chainId` code examples

<CodeGroup>
```javascript ethers.js
const { JsonRpcProvider } = require("ethers");

const provider = new JsonRpcProvider("CHAINSTACK_NODE_URL");

async function call() {
  const result = await provider.send("eth_chainId", []);
  console.log(result);
}

call();
```

```python web3.py
from web3 import Web3

node_url = "CHAINSTACK_NODE_URL"
web3 = Web3(Web3.HTTPProvider(node_url))

result = web3.provider.make_request("eth_chainId", [])
print(result)
```

```shell cURL
curl -X POST "CHAINSTACK_NODE_URL" \
  -H "Content-Type: application/json" \
  --data '{
    "jsonrpc": "2.0",
    "method": "eth_chainId",
    "params": [],
    "id": 1
  }'
```
