Robinhood Chain API method that returns the number of the most recent block.

## Parameters

None

## Response

* `quantity` — the current block number the client is on

## `eth_blockNumber` code examples

<CodeGroup>
```javascript ethers.js
const { JsonRpcProvider } = require("ethers");

const provider = new JsonRpcProvider("CHAINSTACK_NODE_URL");

async function call() {
  const result = await provider.send("eth_blockNumber", []);
  console.log(result);
}

call();
```

```python web3.py
from web3 import Web3

node_url = "CHAINSTACK_NODE_URL"
web3 = Web3(Web3.HTTPProvider(node_url))

result = web3.provider.make_request("eth_blockNumber", [])
print(result)
```

```shell cURL
curl -X POST "CHAINSTACK_NODE_URL" \
  -H "Content-Type: application/json" \
  --data '{
    "jsonrpc": "2.0",
    "method": "eth_blockNumber",
    "params": [],
    "id": 1
  }'
```
